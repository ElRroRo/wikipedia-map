Wikipedia Map

A tool for exploring the world’s knowledge by making connections between topics.

1. Search: Input the name of a Wikipedia article. Compare multiple topics by pressing , Tab or Enter after each one.

2. Generate: Submit your query or adjust the auto-expand depth. Wikipedia Map will create and expand a root node for each input topic.

3. Explore: Click any node to open a context menu where you can read the full article or dynamically expand that specific branch even deeper.
